# PhotoMetadataManager (图片无损隐私清理工具)

这是一个基于 .NET 10 和 ExifTool 开发的**图片隐私属性无损清理工具**。

它最大的特点是**绝对无损**：在抹除 GPS、版权、作者等敏感信息时，只在二进制字节流层面进行精准擦除，**绝对不会对照片像素进行二次压缩（不伤画质），并且能完好保留照片自带的星级、相机私有数据和修改日期**。

---

## 🚀 如何使用

本项目已打包为**单文件可执行程序**，所有的运行依赖和 ExifTool 组件均已内嵌。你不需要配置任何环境变量。

### 第一步：初始化配置文件
直接双击运行主程序 `PhotoMetadataManager.exe`。
* 程序在首次运行时，如果发现同级目录下没有配置文件，会自动在旁边释放出一个默认的 `appsettings.json`。
* 看到提示后，你可以按任意键退出，或者直接开始修改该配置文件。

### 第二步：启动扫描
再次双击或在命令行运行主程序，根据控制台提示进行交互输入：
1. **输入要扫描的文件夹路径**（支持直接将文件夹拖入控制台。如果直接回车，则默认使用配置文件里的路径）。
2. **输入过滤条件**（例如：只检查某一天拍摄的照片，输入 `20260516-*`；直接回车则扫描所有图片 `*.*`）。
3. **选择是否递归子目录**（输入 `y` 检查子文件夹，回车默认不检查）。

随后，程序会极速扫描该目录，并在屏幕上实时输出彩色表格。所有包含敏感信息（如 GPS）的行，其对应单元格会**红底高亮**显示。

### 第三步：交互式无损清理
扫描结束后，如果发现有待清理的文件，程序会自动进入“清理阶段”：
* **查看图片 (`o`)**：输入 `o` 并回车，会立刻调用 Windows 默认的看图软件打开该图片，方便你确认这是哪张照片。看完后，控制台会继续等待你的清理指令。
* **确认清理 (`y`)**：输入 `y` 并回车，控制台会用黄字**实时显示正在剥离的属性**（如 `正在清理: 定位`），并秒级完成无损抹除。
* **跳过该文件 (`n` 或直接回车)**：输入 `n` 或直接敲回车，将保留数据并移动到下一个待处理文件。

## 🛠️ 如何通过 `appsettings.json` 修改配置

本程序所有的展示和清理规则均由 `appsettings.json` 决定。你可以用记事本任意修改它：

### 1. 修改默认扫描路径与参数
如果你每次都扫描同一个文件夹，可以直接修改 `"Scan"` 块，免去每次运行都要手动输入的麻烦：
```json
"Scan": {
  "FolderPath": "D:\\MyPhotos", // 默认扫描路径
  "Filter": "*.*",             // 默认过滤条件
  "Recurse": false             // 默认不扫描子目录
}
```

### 2. 让文件名显示得更完整（调整列宽）
如果在扫描时文件名被截断（显示为 `..`），只需找到 `"Name": "FileName"` 那一行，把它的 **`"Width"`** 数值调大（例如改成 `50` 或 `60`）即可：
```json
{
  "Name": "FileName",
  "DisplayName": "文件名",
  "Width": 50, // 调大这个数值可以完整显示长文件名
  "ExifTag": "FileName",
  "IsClearTarget": false,
  "ClearCommands": []
}
```

### 3. 配置你想清理的其他列（不仅是定位）
默认配置下，只有**“定位”**列被设为了清理目标（`"IsClearTarget": true`）。
如果你想在清空定位的同时，连同照片的 **“版权”**、**“作者”** 或 **“星级”** 一并抹除：
1. 在配置文件中找到对应的那一列（例如 `"Name": "Copyright"`）。
2. 将其中的 **`"IsClearTarget"` 修改为 `true`**。
3. 再次运行程序。此后只要扫描到该列有数据的照片，该属性就会标红高亮，并在你按下 `y` 确认后一并被无损清空！
